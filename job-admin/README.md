# 前后端分离-毕业生就业服务平台

## 项目简介
该项目是基于SpringBoot+Redis+Vue.js实现，采用采用JWT实现前后端分离验证，项目主要分为前台网站和后台管理系统。

## 技术架构
- 后端框架：SpringBoot 2.3
- 数据库：MySQL 5.7
- 缓存数据库：Redis
- 前后端验证技术：JWT、token
- 前端框架：Vuejs、ElementUI

## 运行前台项目
```bash
npm run serve
```
